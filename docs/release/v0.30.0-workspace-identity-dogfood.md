## False-positive review

A local, source-stratified review covered 32 groups (64 rows) across eight sources. The grouping contract already requires identical normalized content and the same session/workspace/source tuple; every reviewed group was within 10 seconds and 24/32 were within 5 seconds (maximum 6.77 seconds). No structural contradiction to replay was found. Intentional repeated content inside that window cannot be ruled out without semantic body review, so automatic cleanup remains prohibited and the existing quarantine apply/restore path remains explicit.

## Release decision

The overall historical candidate rate is below 1%, but Antigravity is far above target and the copied pre-ledger store cannot provide a post-rollout exact-delivery sample. #1465 was filed as a release blocker to add an end-to-end Antigravity Stop replay regression and record a fresh v0.30 ledger sample before release.
