# v0.30.0 上限付きヒューリスティックレポート dogfood

日付: 2026-07-22

Issue: #1467

## 方法

インストール済み v0.29.0 バイナリで、稼働中のローカルストアを `/private/tmp/traceary-v030-1467-dogfood.db` へ整合性のある形でバックアップしました。Issue #1467 のバイナリが migration したのはこのコピーだけで、稼働中の行は変更していません。変更前後のバイナリで、同じ 3.06 GiB のコピーに対して `report workspace-identity --json` を実行し、`/usr/bin/time -l` で測定しました。

最適化後のバイナリは次の 2 経路を測定しました。

1. ヒューリスティックを要求しない、本文非参照の既定レポート
2. `--include-heuristic --heuristic-limit 5000` を指定した明示的な上限付き測定

この文書に記録するのは JSON の集計フィールド、経過時間、最大常駐メモリだけです。イベント本文、生の行、credential、識別子サンプルは commit も外部レビュアーへの送信もしていません。

## コピーしたストアの母集団

- event: 212,023 件
- 履歴上の対象 hook prompt/transcript: 45,150 件
- 既存の dedupe archive 行: 0 件
- 実行時の配送試行: 0 件（元の稼働ストアが v0.30 台帳導入前のため）

## 結果

| 経路 | ヒューリスティック状態 | 読み込んだ本文 | 候補 | 経過時間 | Peak RSS |
|---|---:|---:|---:|---:|---:|
| 変更前の既定 (`a948f4ac` の `main`) | 暗黙の complete | 45,150 | 398 | 22.06 秒 | 733,560,832 bytes (699.6 MiB) |
| #1467 の既定 | `not_requested` | 0 | 0 | 0.01 秒 | 30,752,768 bytes (29.3 MiB) |
| #1467 の明示上限 5,000 | `partial` | 5,000 / 45,150 | 172 | 0.59 秒 | 179,486,720 bytes (171.2 MiB) |

このキャッシュ済みローカル比較では、既定経路は 2,206 倍高速で、Peak RSS は変更前の約 4.2% でした。この比率は今回のコピーしたストアに限る値です。絶対値はファイルシステムキャッシュと本文サイズに依存します。

網羅率クエリも、全イベントとワークスペース観測を相関走査する形から、インデックスを使う observation-to-event join へ変更しました。この修正前には、意図的に古い開発中スキーマのコピーに対する cold 実行が、後段で失敗するまでに 81.41 秒を消費しました。この失敗実行は診断証跡としてのみ扱い、比較表には使っていません。

## 状態と安全性の確認

- 既定 JSON は `measurement_state=not_requested` を返し、exact delivery とワークスペース metric は通常どおり出力しました。
- 明示的な上限付き JSON は `measurement_state=partial`、`eligible_count=45150`、`scanned_count=5000` を返しました。
- CLI テストは `not_requested`、`partial`、`complete`、`failed` の 4 状態をすべてカバーします。
- failed 状態のテストでは、ヒューリスティックエラーを返す前に exact delivery フィールドが出力されることを確認します。
- ストア管理ユースケースは、走査上限と `Apply=true` の併用を拒否します。
- 既存の store dedupe dry-run/apply/restore テストは変更せず、すべて成功します。
- レポート実行後も、コピーしたストアは event 212,023 件、dedupe archive 0 件のままでした。

## リリース判断

#1467 のブロッカー条件を満たしました。通常の workspace-identity 自己診断は本文を読み込まず、明示的な履歴測定には上限と状態があり、exact delivery は独立したままです。コピーした大規模ストアでの時間・メモリ証跡も v0.30.0 の要件を満たします。
